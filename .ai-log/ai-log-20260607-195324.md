# AI Work Log

## Objectives

Cải tiến tính năng luyện tập từ vựng:
1. Thêm chiều luyện tập (Anh → Việt / Việt → Anh) cho màn hình Trắc nghiệm (multi-choice).
2. Thêm chế độ luyện tập mới: "Câu văn" (sentence) sử dụng AI để tạo 10 câu hỏi từ danh sách tối đa 50 từ vựng random shuffled.
3. Chế độ Câu văn cũng hỗ trợ 2 chiều: Anh → Việt (trắc nghiệm) và Việt → Anh (điền vào chỗ trống có gợi ý ký tự).

## Plan

1. Cập nhật các types (thêm type `sentence`, `StudyDirection`).
2. Cập nhật config consts (labels).
3. Cập nhật `vocabulary-study.tsx` để thêm dropdown chọn chiều (direction) và hỗ trợ chế độ Câu văn (validate API Key).
4. Sửa `multi-choice.tsx` để hiển thị UI và đáp án theo chiều được chọn:
   - Anh → Việt: từ TA -> hiển thị nghĩa TV.
   - Việt → Anh: nghĩa TV -> hiển thị từ TA.
5. Tạo `sentence.tsx`:
   - Gửi request qua `useAI` tạo 10 câu hỏi với cấu trúc JSON (englishSentence, vietnameseSentence, wrongOptions).
   - Anh → Việt: dạng trắc nghiệm 4 đáp án tiếng Việt.
   - Việt → Anh: dạng fill-blank, người dùng nhập câu tiếng Anh dựa trên gợi ý các ký tự.
6. Cập nhật `guide.md` phản ánh kiến trúc mới.
7. Xử lý thiếu package `@radix-ui/react-switch` và import sai của Shadcn Switch.
